# 01 项目启动说明

## 1. 项目名称
校园活动报名与签到管理系统

## 2. 项目目标
- 支持活动管理人员发布活动信息并维护报名时间、人数上限和活动地点
- 支持学生查看活动并完成在线报名
- 支持活动组织者对报名名单进行审核与查看
- 支持现场二维码签到，并形成签到结果统计
- 支持活动结束后导出基本统计结果

## 4. 项目边界

### 纳入范围
- 活动信息发布与维护
- 学生报名信息提交
- 报名名单查看与审核
- 签到记录与签到状态管理
- 签到统计结果导出

### 不纳入范围
- 复杂支付功能
- 与真实校园统一身份认证系统的深度对接
- 高并发优化与正式部署方案
- 移动端 App 正式开发
